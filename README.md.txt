# Customer Churn Prediction using ML Pipeline

## 📌 Objective
Build a production-ready machine learning pipeline to predict customer churn.

## 📊 Dataset
Telco Customer Churn Dataset

## ⚙️ Methodology
- Data preprocessing using Pipeline
- Feature scaling and encoding
- Model training using:
  - Logistic Regression
  - Random Forest
- Hyperparameter tuning using GridSearchCV

## 📈 Results
- Achieved good accuracy on test data
- Model evaluated using:
  - Accuracy
  - Classification Report

## 💾 Model Export
Final pipeline saved using joblib for reuse

## 🚀 Technologies Used
- Python
- Scikit-learn
- Pandas

## 📌 Conclusion
The pipeline provides a reusable and scalable solution for churn prediction.
